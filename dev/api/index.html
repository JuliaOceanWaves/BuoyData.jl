NDBC.available — Function
available()
available(buoy)

Return a DataFrame listing buoy-year combinations where all five spectral files exist (swden, swdir, swdir2, swr1, swr2). When a buoy is provided, returns only rows for that buoy.

NDBC.request — Function
request(buoy, year, b_file=false)

Download and parse the full directional wave spectrum for a buoy and year. Returns an AxisArray indexed by time, frequency, and parameter.
